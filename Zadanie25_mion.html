Akapit 1-centrowany

HTML powstał w 1990 roku i został stworzony przez W3C

HTML (ang. HyperText Markup Language, hipertekstowy język znaczników) – język znaczników stosowany do tworzenia dokumentów hipertekstowych.

HTML pozwala opisać strukturę informacji zawartych wewnątrz strony internetowej, nadając odpowiednie znaczenie semantyczne poszczególnym fragmentom tekstu

css - kaskadowe arkusze stylów

css powstał w 1994 przez W3C

CSS to język, który służy do stylizowania dokumentów HTML.
