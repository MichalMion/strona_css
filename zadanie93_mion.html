ul -> (Lista nieuporządkowana)
{
list-style: none; -> (usuwa domyślne kropki lub numerację z elementów listy.)
margin: 5px; -> (dodaje margines 5 pikseli wokół listy.)
padding: 0px; -> (usuwa wewnętrzny odstęp (padding) z elementów listy.)
display: block; -> Ustawia element jako blokowy
width: 150px; -> Określa szerokość na 150 pikseli.
ul li -> lista uporzadkowana i nieuporzadkowana
{
display: block; -> Ustawia elementy listy jako blokowe.
list-style: none; -> Wyłącza domyślne oznaczenie punktami.
margin: 0px; -> Wyłącza margines zewnętrzny.
padding: 0px; -> Wyłącza wewnętrzny padding.
}
ul a:link, -> (lista nieuporządkowana -> stylizacja linku który nie był jeszcze kliknięty)
ul a:visited -> (lista nieuporządkowana -> stylizacja linku który był kiedyś kliknięty)
{
display: block; -> element będzie wyświetlony w bloku (odstęp z góry iz dołu)
color: #005500; -> (ustawia kolor tekstu linków )
text-decoration: none; -> (usuwa podkreślenie z linków)
border: 2px outset #cccccc; -> Dodaje obramowanie o szerokości 2 pikseli, styl “outset” i kolor szary.
padding: 5px; -> Ustala wewnętrzny padding na 5 pikseli.
width: 120px; -> Określa szerokość na 120 pikseli.
}
ul a:hover -> (lista nieuporzadkowana -> co się stanie po najechaniu kursorem )
{
color: #ff00ff;
text-decoration: none; -> Wyłącza podkreślenie linków.
border-style: inset; -> Zmienia styl obramowania na “inset”.
padding: 7px 3px 3px 7px; -> góra i zgodnie ze wskazówkami zegra.
}
